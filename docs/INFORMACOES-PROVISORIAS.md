# Informações provisórias

- local do festival;
- aprovação ou financiamento pelo FMIC;
- escolas participantes;
- programação;
- palestrantes;
- patrocinadores;
- expositores;
- entrada gratuita para todo o festival;
- atrações e experiências específicas.

- data e status;
- local;
- vídeos;
- parceiros;
- Instagram;
- destino dos formulários;
